# trmnl-osrs-ge-price-tracker
Trmnl price tracker for the Grand Exchange

## How to use
Just grab an Item ID from https://secure.runescape.com/m=itemdb_oldschool/api/catalogue/detail.json?item=22486.  By default the item is Scythe of vitur (uncharged).


## installing the plugin
You can install the [plugin](https://trmnl.com/recipes/265824) from TRMNL plugin market place.
